Comandos GitHub

  • git init -> Iniciar um repositório no computador.
  • git add . -> Adicionarmos as alterações dos aqrquivos.
  • git status -> Verificar as alterações que foram feitas.
  • git commit -m "Texto Do commit" -> Explicar o que está adicionando.
  • git push origin main -> Enviando as alterações para o repositório da origem.
